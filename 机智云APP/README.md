# 机智云 APP 使用说明

本项目通过 ESP8266 接入机智云平台，用于在手机端查看农业大棚环境数据并远程控制设备。

## 支持功能

- 查看温度、湿度、CO2 浓度、光照强度、土壤湿度等数据。
- 远程控制水泵、风扇、蜂鸣器和补光 LED。
- 支持一键控制相关执行器。

## 使用说明

1. 给 STM32 主控板和 ESP8266 模块上电。
2. 根据机智云配网流程，将 ESP8266 连接到可用 WiFi。
3. 在 APP 中绑定设备。
4. 进入设备页面查看实时数据，并测试水泵、风扇、LED 等远程控制功能。
